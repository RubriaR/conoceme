Description
Conóceme is an interactive electronic literature piece that invites readers to engage with common stereotypes about migrants. Set against an agricultural landscape, the screen is initially covered with negative labels. As the user clicks on them, they transform into personal truths. Once all the prejudices are cleared, a final text is revealed.


👉 View the project: https://rubriar.github.io/conoceme/

Artistic Statement
Conóceme uses digital interaction to explore how we perceive the migrant experience. The piece requires the user to click the hostile words scattered on the screen to uncover the human reality behind the stereotypes. By transforming these labels into simple statements about work, fear, and survival, the narrative encourages readers to look past generalizations and reflect on our shared humanity.
